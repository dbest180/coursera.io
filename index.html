How to Submit Your GitHub Pages Link to Coursera

This guide explains how to take your HTML project, put it online with GitHub Pages, and submit the live link to Coursera.

1. Put Your Project on GitHub

  • Create a GitHub account if you don’t have one.
  • Create a new public repository.
  • Name it anything you like.
  • Upload your index.html file and any images.

2. Turn On GitHub Pages

  • Open your repository.
  • Go to Settings.
  • Find the section called Pages.
  • Under “Source,” choose:
    • Deploy from a branch
    • Branch: main
    • Folder: / (root)
  • Save your settings.

GitHub will create your live webpage. It appears in a green box at the top of the Pages screen.

3. Copy Your Live Link

Your link will look like this:

https://your-username.github.io/your-repository-name/

This link is what Coursera needs.

4. Submit It to Coursera

  • Go to the assignment submission page on Coursera.
  • Paste your GitHub Pages link into the text field.
  • Make sure the link opens your webpage before submitting.
  • Submit your assignment.

All Done

Your HTML project is now live on the internet and submitted to Coursera. If you update your files later, GitHub Pages will refresh your page automatically.
