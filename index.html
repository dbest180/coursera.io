Submit to Coursera

Publish your HTML project with GitHub Pages and submit the live link. Follow the steps below.

1. Put Your Project on GitHub

  • Create a GitHub account if you don't already have one.
  • Create a new public repository.
  • Name it anything you like.
  • Upload your index.html file.
  • Upload any images your page uses.

2. Turn On GitHub Pages

  • Open your repository.
  • Go to Settings.
  • Scroll down and select Pages.
  • Under "Source," choose Deploy from a branch.
  • Select the main branch.
  • Select the root folder.
  • Save your changes.

GitHub will generate your live webpage in a few seconds.

3. Copy Your Live Link

Your link will look something like this:

https://your-username.github.io/your-repository-name/

This is the exact link Coursera requires.

4. Submit the Link

  • Open your Coursera assignment submission page.
  • Paste your GitHub Pages link into the text field.
  • Make sure the link loads your webpage properly.
  • Submit your assignment.

All Done

Your HTML page is now live, and your Coursera assignment is submitted. If you later edit or replace your files, GitHub Pages will update your site automatically.
