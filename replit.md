# Submit to Coursera - GitHub Pages Guide

## Overview
This is a static HTML website that provides step-by-step instructions for publishing HTML projects using GitHub Pages and submitting them to Coursera. The site features a modern, glassmorphic design with animated backgrounds and gradient effects.

## Project Structure
- `index.html` - Main HTML page with instructional content
- `style.css` - Stylesheet with modern design and animations
- `favicon-master.png` - Favicon image (currently unused)
- `README.md` - Project documentation

## Technology Stack
- Pure HTML5, CSS3
- Google Fonts (Inter & Space Grotesk)
- Python's built-in HTTP server for local development

## Current State
The website is fully functional and displays:
1. Instructions for putting projects on GitHub
2. Steps to enable GitHub Pages
3. How to copy the live link
4. Submission process for Coursera
5. Completion confirmation

## Development
- Server runs on port 5000 using Python's HTTP server
- Static files are served directly from the root directory
- No build process required
